# Georeactor Explainer Video — Production Plan

## Global Visual Style

| Dimension | Specification |
|-----------|--------------|
| **Sub-genre** | Cinematic scientific illustration, NASA-style space visualization |
| **Rendering + Line** | Photorealistic 3D rendering with volumetric lighting, no outlines |
| **Color + Lighting** | Deep ocean blues, molten orange/gold cores, emerald greens, Cherenkov cyan glows. Dramatic rim lighting, volumetric god rays |
| **Detail density** | Highly detailed environments, cinematic depth of field |

## Recurring Elements

| Element | Description |
|---------|-------------|
| **Earth** | Photorealistic planet Earth, sometimes shown as cross-section revealing glowing inner core |
| **Core** | Brilliant white-hot center transitioning to molten orange, surrounded by deep red outer core |
| **Mantle** | Dark amber to deep red flowing material with convection currents |
| **Crust** | Thin dark layer on surface with visible tectonic plates |
| **Nuclear glow** | Cherenkov-blue radiation glow emanating from core reactions |
| **Magnetic field lines** | Cyan/blue luminous lines arcing from pole to pole |

## Voice Profile

| Field | Value |
|-------|-------|
| **Type** | Off-screen narrator |
| **Gender** | Male |
| **Tone** | Calm, authoritative, slightly awestruck — like a documentary narrator discovering something profound |
| **Pace** | Moderate, with pauses for dramatic effect |
| **Language** | English |

## BGM

| Field | Value |
|-------|-------|
| **Source** | Separate (royalty-free) |
| **Genre** | Ambient cinematic, building orchestral |
| **Instruments** | Deep synth pads, subtle strings, low percussion |
